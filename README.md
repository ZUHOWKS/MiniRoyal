# Mini-Royal Minecraft Plugin (1.9.X - 1.X)
Le mode de jeu le plus populaire de cette décennie est enfin disponible sur Minecraft en version mini-jeux ! Profitez d'un système de cosmetique, avec des planneurs, des pioches et pleins d'autres items fou !


## Pré-requis
Ce plugin s'implémente sur un système de serveur à la demande avec la configuration d'un script de réinitialisation de map et de rédemarrage automatique du serveur. 

## Compatibilité 
Ce plugin a été codé selon l'API spigot 1.9.X et est donc compilé avec Java 8. Pour les versions de Minecraft supérieurs ou égale à la 1.16.X, le plugin est compilé avec Java 17. 
En résumé il existe une version du plugin pour la 1.9.X - 1.15.X et une version compatible de la 1.16.X - 1.20.X.
